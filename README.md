# 💑 Хосын хоолны дэвтэр

Хоёулаа үдийн хоолоо үнэлж, Gemini AI-ийн зөвлөгөөгөөр дараагийн хоолоо сонгодог веб апп.

## Ашиглах заавар

`index.html` файлыг хөтөч дээр нээнэ.

## Боломжууд

- Хоёр хүний тусдаа үнэлгээ (ерөнхий, амт, хэмжээ, орц)
- Сэтгэл санааны feedback (байдал + ажлын ачаалал)
- Хоолны онцлог keyword-ууд
- Gemini AI — тухайн хүний хувийн дүгнэлт
- Дараагийн хоолны хамтарсан зөвлөгөө
- Хоолны дэвтэр (LocalStorage)

## API

Gemini 1.5 Flash (Google AI Studio үнэгүй tier)
